Meus Softwares

  • Figma

    O Figma é uma ferramenta de design colaborativo baseada na web, ideal para criação, prototipagem e compartilhamento de interfaces de usuário.

  • VisualEyes

    VisualEyes é uma plataforma de análise de dados e visualização, útil para compreender e comunicar informações complexas de forma visual.

  • Sketch

    O Sketch é um aplicativo de design vetorial focado em interface de usuário e gráficos digitais, amplamente utilizado por designers.

  • Ilustrator

    O Adobe Illustrator é um software de design gráfico vetorial usado para criar ilustrações e gráficos precisos e escaláveis.

  • Photoshop

    O Adobe Photoshop é um poderoso software de edição de imagens usado para criar, retocar e manipular fotos digitalmente.

Jornada de conhecimento

UX Design

Iniciei minha carreira como primeiro UX Designer, focando em aprimorar a experiência do usuário em produtos digitais. Meu trabalho consistia em compreender as necessidades e desejos dos usuários para criar interfaces intuitivas e eficazes.

Viés Cognitivo e Neuromarketing em Design

Aprofundei meus conhecimentos ao explorar viés cognitivo e neuromarketing no design. Apliquei insights psicológicos para criar interfaces que cativam e influenciam positivamente o comportamento do usuário, aprimorando a eficácia das estratégias de marketing.

UI Design

Posteriormente, migrei para o papel de UI Designer, concentrando-me na estética e na usabilidade das interfaces digitais. Trabalhei para criar designs visualmente atraentes que também oferecessem uma experiência de usuário excepcional.

Mentoria para Projetos em Empresas e Startups

Atualmente também ofereço mentoria especializada para projetos de empresas e startups. Compartilho minha experiência e conhecimento para ajudar equipes a alcançar seus objetivos, impulsionando o sucesso de seus projetos e negócios.
